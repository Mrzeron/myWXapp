# 花礼网微信小程序
## 主要实现以下简单功能

1. 首页面布局实现
2. 首页面到底部自动加载下一屏
3. 首页面下滑超过500px 出现返回顶部按钮点击可返回顶部
4. 首页轮播图实现
5. 底部5个导航点击切换的效果实现
6. 上拉刷新效果的实现
7. 首页热卖鲜花区域点击可跳转详情页
8. 详情页布局的基本实现
9. 详情页关注该商品功能的实现
10. 首页面和详情页数据交换的实现
